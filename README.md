# Python DevOps 实验室

基于《基于Python的DevOps》一书的学习项目。

## 项目结构

- `src/` - 源代码目录
- `tests/` - 测试代码目录
- `scripts/` - 脚本工具目录
- `config/` - 配置文件目录
- `docs/` - 文档目录

## 环境搭建

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行测试：
   ```bash
   pytest
   ```

3. 运行脚本：
   ```bash
   python scripts/hello.py
   ```
