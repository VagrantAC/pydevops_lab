# Lab 1: Python基础与环境配置

## 实验目标
- 熟悉Python开发环境
- 掌握基本的Python语法
- 理解虚拟环境的使用
- 学习基本的调试技巧

## 实验步骤
1. 检查Python版本
2. 创建虚拟环境
3. 安装依赖包
4. 编写基础Python代码
5. 运行测试

## 运行方式
```bash
# 运行主脚本
python labs/lab1/scripts/main.py

# 运行测试
pytest labs/lab1/tests/
```

## 预期输出
- Python版本信息
- 环境配置状态
- 基础代码执行结果

## 学习要点
- Python环境变量
- 包管理工具pip
- 虚拟环境venv
- 基础数据类型
- 控制流程
